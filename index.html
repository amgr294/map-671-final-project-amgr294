Housing Cost and Tree Canopy Coverage

Visual Relationship Between Housing Cost and Tree Canopy Coverage in Seattle, Washington

Authored by: Ashley Grospitch for MAP 671 Fall 2025

Background and Project Overview

This map explores the relationship between tree canopy coverage and housing costs in Seattle, Washington through visual comparison. Tree canopy coverage is represented by color density, with darker greens indicating higher density, and housing cost is represented by average cost per square foot.

The two datasets used in this analysis come from the Seattle Open Data Portal. The first dataset, "Seattle Tree Canopy 2021" was used to show the locations of tree canopy across the metropolitan area, mapped in 2021 using remotely sensed data. The second dataset, "Single-Family Home Sale Prices by Census Tract", was used to show the average price per square foot by census tract. This dataset was specific to single-family homes, meaning the analysis is only applicable to this housing type.

Key Findings

Key findings from the map include that tree canopy does appear to be loosely correlated with housing cost. Parts of west Seattle have a higher tree canopy density and also a higher price per square foot. Areas of downtown have less tree canopy and a lower price per square foot despite being in the downtown cooridor. This initial analysis provides a start in determining the relationship between housing price and tree canopy density and would need additional analyses to determine a statistical relationship.

Mapmaking

To make the tree canopy map, I downloaded and added the data as a vector layer. I then adjusted the symbology, changing the color to represent trees and render settings to "multiply," which makes the map appear darker in areas of higher tree canopy density. To make the housing price map, I categorized layer symbology by the price per square foot and created a graduated color ramp where the darker the color the higher the price per square foot ($).

The original projection was in EPSG: 2926, which I also kept as the final projection because this is the projected coordinate system in feet for the north Washington area. QGIS 3.34.2 was used to create this map.
